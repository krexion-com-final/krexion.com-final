Install Karne Ka Tareeqa

1
ZIP Download Karein
Upar wala "Download ZIP" button click karein. ~10 MB file save hogi.
2
Extract Karein
ZIP file par right-click → "Extract All" → Desktop pe extract karein.
3
Installer Run Karein
Extracted folder ke andar Krexion-ULTIMATE-INSTALL.bat file ko double-click karein.
4
UAC Popup → Yes
Windows admin permission maange to "Yes" click karein.
5
Wait Karein (20-30 min)
Coffee piyen ☕. Installer sab kuch khud kar lega. End mein browser khud khulega.
