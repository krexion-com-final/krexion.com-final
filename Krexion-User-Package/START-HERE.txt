                    KREXION - YAHAN SE START KAREIN
════════════════════════════════════════════════════════════════

  ZAROORI: ZIP file ke ANDAR se INSTALL.bat NA chalayein!
           Pehle file ko PROPERLY extract karein.


  STEP 1 - ZIP EXTRACT KAREIN:
  ─────────────────────────────
  1. "Krexion-User-Package.zip" pe RIGHT-CLICK karein
  2. "Extract All..." click karein
  3. Phir "Extract" button click karein
  4. Naya folder khulega — us folder ke ANDER jayein


  STEP 2 - INSTALLER CHALAYEIN:
  ─────────────────────────────
  1. "INSTALL.bat" file pe DOUBLE-CLICK karein
  2. UAC popup aaye to "Yes" / "Haan" click karein
  3. Bas! Installer khud sab kuch handle kar lega.

  Tip: Agar popup turant band ho jaye, INSTALL.bat pe
       RIGHT-CLICK kar ke "Run as administrator" select karein.


  STEP 3 - WAIT KAREIN:
  ─────────────────────────────
  - Installer 20-30 minute lega (ek-time setup hai)
  - Window mein progress dikhata rahega
  - Internet stable rakhein
  - PC sleep mein na jaye (Settings → Power → Never sleep recommended)


  STEP 4 - LOGIN KAREIN:
  ─────────────────────────────
  Install hone ke baad browser khud khul jayega
  https://krexion.com/login pe.

  Welcome email mein jo email + password mile, wahi daalein.
  License key bhi same email mein milegi.


  License nahi hai?
  ─────────────────
  https://krexion.com/pricing pe jaa kar khareedein
  (USDT-TRC20 se pay karein, 30 min mein email pe milegi)
