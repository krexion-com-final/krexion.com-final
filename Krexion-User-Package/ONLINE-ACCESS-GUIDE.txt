YEH FILE INSTALL KE BAAD PADHEIN
==================================


KAB ZAROORAT HAI?
===================

  - Ghar ki PC pe Krexion chal raha hai
  - Aap office mein ya kahin aur jana hai
  - Wahan se bhi Krexion use karna hai
  - Mobile pe Krexion kholna hai


KYA KAREIN?
=============

  Step 1: Confirm karein Krexion chal raha hai
    - Browser mein http://localhost:3000 kholein
    - Agar khulta hai = OK, agar nahi to LOCAL-START.bat chalayein

  Step 2: Folder kholein
    - File Explorer mein jayein
    - C:\krexion folder kholein

  Step 3: GO-ONLINE.bat double-click karein
    - 30-60 second wait karein
    - Pehli baar cloudflared.exe download hoga (~25 MB)
    - Console window khulegi

  Step 4: Public URL note karein
    - Console mein URL likha hoga jaise:
      https://abc-xyz-123.trycloudflare.com
    - Yeh URL copy karein

  Step 5: Mobile/doosri PC pe use karein
    - URL paste karein browser mein
    - Login karein apne email + password se
    - Krexion chalu!


IMPORTANT NOTES
=================

  ! Console window BAND NA KAREIN
    Window band hone par URL dead ho jata hai
    Mobile pe access bhi band ho jata hai

  ! Har baar URL CHANGE hota hai
    Quick Tunnel ka URL har baar new hai
    Doosri baar GO-ONLINE.bat chalayein = naya URL

  ! Free hai, koi cost nahi
    Cloudflare ka free service hai
    Koi signup ya domain ki zaroorat nahi

  ! PC band hone par offline ho jata hai
    Public URL sirf jab PC on + Krexion chal raha + 
    GO-ONLINE.bat window khuli ho


MOBILE PE QR CODE SE
======================

  Console window khulne ke baad:
    1. URL ke saath QR code bhi dikhega
    2. Mobile se QR scan karein (Camera app se)
    3. URL khud kholne ka option aayega
    4. Click karein -> Krexion mobile pe khulay


WhatsApp PE URL SHARE KARNE
=============================

  Console window mein WhatsApp share button hai
    1. Button click karein
    2. WhatsApp khulay
    3. Contact select karein
    4. URL send ho jayegi


PERMANENT URL CHAHIYE?
========================

  Agar har baar same URL chahiye:
    1. Aap (customer) hum (admin) se contact karein
    2. Hum permanent setup help karenge
    3. Cloudflare Named Tunnel se permanent URL
    4. Cost: Free (sirf Cloudflare account)
    5. Setup time: 15 min


URL SAFE HAI?
================

  Haan, bilkul safe:
    - HTTPS encryption hota hai
    - Sirf aap ki PC se traffic flow hota hai
    - Data sirf aap ke aur Cloudflare ke beech mein
    - PC band karne par URL bhi band

  Lekin sharing pe care karein:
    - URL kisi ko bhejen to wo bhi access kar sakta hai
    - Sirf trusted logon ko bhejein
    - Use khatm ho to GO-ONLINE.bat console close kar dein



KAB OFFLINE HOTA HAI?
========================

  - Console window close kiya
  - PC band ho gayi
  - Internet disconnect ho gaya
  - C:\krexion ka Krexion service stop kiya
  - PC sleep / hibernation mein gayi
