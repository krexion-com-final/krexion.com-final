RealFlow Admin Toolkit

Local browser tool — koi data internet pe nahi jaata. Sab kuch aap ke browser ke andar compute hota hai.

🔑 Tool 1 — Activation Key Hash Generator

Nayi activation key banao → SHA-256 hash niche ayega → installer mein paste karo

-
Kahan paste karein: RealFlow-INSTALL.bat file mein line dhoondein:
set "EXPECTED_HASH=..." — purani hash ko nayi se replace karein → commit → done.

⏰ Tool 2 — Expiry Date Generator

Aaj se kitne din baad expire karna hai? Niche format installer mein paste karna hai.

-
Readable date yahan ayega...
Kahan paste karein: RealFlow-INSTALL.bat mein line dhoondein:
set "EXPIRY_DATE=20260710" — number replace karein → commit → done.

🌐 Tool 3 — Kill-Switch (REVOKE / ACTIVATE)

GitHub mein .installer-status file ko set karne ke ready-to-copy commands.

REVOKED
ACTIVE
Step-by-step:
1. Yahan click karke .installer-status file kholein
2. ✏️ Pencil icon click karein
3. Andar wala text delete karke upar wala (REVOKED ya ACTIVE) paste karein
4. Niche scroll → Commit changes click karein
5. 30 second mein effect live ho jata hai
