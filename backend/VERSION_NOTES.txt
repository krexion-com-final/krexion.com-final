#   any visible popup / modal / dialog container (role=dialog,

#   cursor:pointer close-Xs) and returns a checklist grouped by
#   popup_index. The customer ticks which button(s) to click and
#   each tick becomes a normal click step via the existing /click
#   endpoint — so RUT replay runs identical to any manual click.

#  Feature B — "Scan" tool (key: S)
# ══════════════════════════════════════════════════════════════════

#   page. Returns `text`, `selector` (CSS), `xpath_stable`, `xpath_abs`,
