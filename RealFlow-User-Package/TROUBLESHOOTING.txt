PROBLEM 1: INSTALL.bat double-click pe kuch nahi hua
======================================================

  Wajah: Windows ne block kar diya hai

  Solution:
    1. INSTALL.bat par RIGHT-CLICK karein
    2. "Run as administrator" click karein
    3. UAC popup pe YES dabayein


PROBLEM 2: "Internet connection nahi hai" error
=================================================

  Wajah: WiFi/Internet kaam nahi kar raha

  Solution:
    1. WiFi check karein
    2. Mobile hotspot try karein
    3. Browser mein https://github.com kholne ki koshish karein
    4. Agar koi website nahi khulti to internet provider se contact
    5. Internet working ho to INSTALL.bat dobara chalayein


PROBLEM 3: "Download failed" error
====================================

  Wajah: Antivirus block kar raha hai ya internet slow

  Solution:
    1. Antivirus 10 min ke liye DISABLE karein
       (Windows Defender, Avast, Kaspersky, etc.)
    2. INSTALL.bat dobara chalayein
    3. Install ke baad antivirus wapas enable kar dein


PROBLEM 4: "Reboot required" message
======================================

  Wajah: Windows features enable hone ke liye reboot chahiye

  Solution:
    1. "Y" press karein ya manually PC restart karein
    2. Restart ke baad login karein
    3. Desktop pe "REALFLOW-RESUME.bat" file mil jayegi
    4. Wo automatically chal jayegi (ya double-click karein)
    5. Installation continue ho jayegi


PROBLEM 5: Docker stuck pe "Starting..." aata hai
====================================================

  Wajah: WSL2 kernel update zaroori hai

  Solution: Installer khud 3 baar fix try karta hai. 
  Agar phir bhi fail:
    1. PC restart karein
    2. Docker Desktop Start Menu se kholein
    3. Whale icon GREEN hone ka wait karein (3-5 min)
    4. INSTALL.bat dobara chalayein


PROBLEM 6: "CPU virtualization disabled" error
================================================

  Wajah: BIOS mein virtualization off hai

  Solution:
    1. PC restart karein
    2. Boot screen pe F2/F10/DEL key press karein
       - Dell: F2 ya F12
       - HP: F10 ya ESC
       - Lenovo: F1/F2/Enter then F1
       - ASUS: DEL ya F2
       - Acer: F2 ya DEL
       - MSI: DEL
    3. BIOS mein dhondein:
       - "Intel Virtualization Technology"
       - "VT-x"
       - "AMD-V"
       - "SVM Mode"
    4. ENABLED set karein
    5. Save & Exit (F10)
    6. PC restart ho jayega
    7. INSTALL.bat dobara chalayein


PROBLEM 7: Install complete, browser nahi khula
=================================================

  Solution:
    Manually browser kholein aur yeh URL paste karein:
    
    http://localhost:3000/register


PROBLEM 8: Register page khuli but submit pe error aaya
========================================================

  Wajah: Backend abhi start nahi hua

  Solution:
    1. 2-3 min aur wait karein
    2. Browser refresh karein (F5)
    3. Phir register try karein


PROBLEM 9: Account ban gaya, login pe "approval pending" ata
==============================================================

  Wajah: Admin ne abhi approve nahi kiya

  Solution:
    1. Admin ko WhatsApp karein
    2. Apna email batayein
    3. Admin approve karega
    4. Phir login try karein


PROBLEM 10: License key galat hai
====================================

  Solution:
    1. Admin se naya key maangein
    2. Pakka key sahi copy-paste karein (spaces na hon)
    3. Phir activate try karein


PROBLEM 11: GO-ONLINE.bat se public URL nahi bana
====================================================

  Solution:
    1. Pehle confirm karein RealFlow chal raha hai
       Browser mein http://localhost:3000 kholein
    2. Agar nahi chal raha:
       - C:\realflow folder kholein
       - LOCAL-START.bat double-click karein
       - 1 min wait karein
       - Phir GO-ONLINE.bat chalayein


PROBLEM 12: Mobile pe URL nahi khulti
========================================

  Solution:
    1. PC pe console window khuli hai? (band na ho)
    2. URL EXACTLY copy karein (no extra spaces)
    3. Mobile pe URL paste karein https://... ke saath
    4. Cellular data on karke try karein
    5. URL refresh ho jaye to GO-ONLINE.bat dobara chalayein



STILL NEED HELP?
==================

  Admin ko yeh information bhejein WhatsApp pe:
  
    1. Screenshot of error
    2. Yeh log files:
       C:\Users\[YOUR-NAME]\AppData\Local\Temp\realflow-install.log
       C:\Users\[YOUR-NAME]\AppData\Local\Temp\realflow-transcript.log
    3. Aap ka Windows version (Settings -> System -> About)
    4. Aap ki RAM (Task Manager -> Performance -> Memory)
  
  Log files kaise milte hain:
    1. Windows Explorer kholein
    2. Address bar mein paste karein: %TEMP%
    3. Enter dabayein
    4. realflow-install.log aur realflow-transcript.log dhondein
    5. Dono files admin ko WhatsApp karein
